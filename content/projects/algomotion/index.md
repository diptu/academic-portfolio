An open-source platform for Machine Learning, Computer Vision, and Autonomous Systems, structured
as a research-readiness foundation rather than a collection of isolated exercises.

- Algorithmic foundations: data structures and algorithms implemented from first principles
- Machine Learning and Deep Learning implementations with architectural rigor, using PyTorch
- Computer Vision experimentation extending toward Edge AI deployment constraints
- Autonomous Systems groundwork: structured for extension into Continual Learning research

**Core stack:** Python, PyTorch, OpenCV, FastAPI, Docker, GitHub Actions
